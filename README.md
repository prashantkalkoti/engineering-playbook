# Engineering Playbook

A collection of architecture decisions, system design explorations, engineering patterns, and practical lessons from designing, building, and operating software systems.

## Topics

- Architecture
- System Design
- Distributed Systems
- Reliability Engineering
- API Design
- Security
- Observability
- Platform Engineering
- Architecture Decision Records (ADRs)

## Structure

- `/architecture` - Architecture patterns and trade-offs
- `/system-design` - System design studies
- `/adr` - Architecture Decision Records
- `/distributed-systems` - Distributed systems concepts
- `/reliability` - Reliability and operational excellence
- `/api-design` - API design patterns and practices
- `/case-studies` - Real-world engineering case studies
